把config文件夹下的index.default.js文件重命名为index.js，并修改文件中的配置。

## 启动

``` shell
npm install
npm run server
```

## 上传文件

上传文件功能的配置不在配置文件中添加，而是在app中添加，进入app后你需要：
1. 添加文件桶 
2. 添加参数QINIU_SECRET_KEY、QINIU_ACCESS_KEY（分别对应你的七牛secretKey和accessKey）
3. 在配置文件中添加七牛的地区（zone）
